# Paper 3 Checklist

## Required Simulation Work

- Generate `N` random processes.
- Use burst times in the range `10..1000`.
- Use arrival times inside `0..M`.
- Vary `N` and `M`.
- Simulate multilevel feedback queue scheduling.
- Search for good `Q1`, `Q2`, `L1`, and `L2` values.
- Add the third-level `T` split between SJF and FCFS.
- Run repeated simulations, ideally 100 or more.
- Study M/M/1 and M/M/S queueing models.
- Compare constant and changing values of `N`, `M`, and `R`.
- Compare different random distributions.

## Required Metrics

- throughput
- turnaround time
- waiting time
- response time
- average queue length
- utilization
- stability

## Required Output

- source code
- generated datasets
- raw simulation outputs
- tables
- plots
- screenshots
- final MS Word report
