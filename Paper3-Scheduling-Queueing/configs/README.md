# Configs

Experiment settings go here. These files should make the simulations reproducible.

Example config topics:

- number of processes
- arrival interval `M`
- burst-time range `10..1000`
- MLFQ values for `Q1`, `Q2`, `L1`, `L2`, and `T`
- number of Monte Carlo runs
- queueing service capacity `R`
- number of servers `S`
- random distribution name
